# Business Requirements Documents (BRD) – Insurance Claim System

This folder contains all Business Requirements Documents (BRD) for the Insurance Claim System. These documents define the business needs, objectives, and high-level requirements for the system from a stakeholder perspective.

---

## Purpose of the BRD

The BRD serves to:

- Capture business goals, functional requirements, and project constraints
- Provide a clear understanding of system objectives to stakeholders, business analysts, and developers
- Serve as a reference throughout the project lifecycle
- Support alignment between business needs and technical implementation

---

## Contents

| File Name | Description |
|---|---|
| `Insurance_Claim_System_BRD.docx` | Main BRD in Microsoft Word format |
| `Insurance_Claim_System_BRD.md` | Markdown version of the BRD for GitHub viewing |
| `Diagrams` | Contains supporting diagrams including flowcharts, UML diagrams, and process models |
